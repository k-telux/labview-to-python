# Changelog

All notable changes follow [Keep a Changelog](https://keepachangelog.com/) and
[Semantic Versioning](https://semver.org/).

## [2.0.0] - 2026-07-15

### Added

- hard industrial rule with Chinese and Japanese summaries;
- performance, displayed-data parity, supervision, and pump-probe case references;
- sanitized real input/output examples from three migration histories;
- Chinese and Japanese project documentation;
- dependency-free repository validator and GitHub Actions workflow;
- issue templates, pull-request template, and citation metadata.

### Changed

- moved the canonical skill to the portable `skills/` layout;
- strengthened proof boundaries for roundtrip fixtures, real UI events, partial
  stops, process ownership, source/EXE lineage, and real hardware;
- expanded installation and distribution guidance.
